CS 210 Project Three - Corner Grocer

Open CornerGrocerProject.sln in Visual Studio and run the project. The program reads CS210_Project_Three_Input_File.txt from the project folder and creates frequency.dat automatically when the program starts.

Menu options:
1. Search for one item and display its frequency.
2. Display all item frequencies.
3. Display a histogram with asterisks.
4. Exit.
